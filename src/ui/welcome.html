qapi

a question and answer API


qapi is locked, meaning you need a code directly from the instance admin to use it. you can request that from them directly if you know the admin. if not, you're welcome to self-host it.


to use the API: read the docs


to use the browser SDK:

const s = document.createElement('script');
s.src = '{{origin}}/sdk.js';
document.head.appendChild(s);
s.onload = () => {
    // run actions with the global `qapi` variable
}

actions include:

qapi.mcq('question', ['opt 1', 'opt 2', 'opt 3']) // -> { answer: 1-based num index, thoughts: string }
qapi.ocr(imageURL) -> { text: string }
qapi.requestCode() // opens the code request modal
qapi.onAuthorize(callback) // when a code is entered and is valid; callback param = code
