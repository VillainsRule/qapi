qapi

a question and answer API


all endpoints require an Authorization header to access, which can be obtained by asking the admin of this instance. available endpoints include:


GET /api/v1/valid - check an Authorization header. 
// queryParams
key=THE-KEY

// response
{
    exists: boolean
}

POST /api/v1/mcq - answer an MCQ. 
// body
{
    question: "What is 2 + 2?",
    answers: ["2", "3", "4", "5"]
}

// response
{
    success: true,
    answer: "4"
}
